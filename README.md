# Kadig - Gestão de Investimentos Inteligente

## Sobre

Kadig é a plataforma completa para gestão de investimentos com IA. Acompanhe sua carteira, analise rentabilidade e alcance seus objetivos financeiros.

## Funcionalidades

- Consolidação de patrimônio (ações, FIIs, renda fixa, criptos, etc.)
- Análise de rentabilidade em tempo real
- Consultoria com IA (Bianca)
- Conexão automática com corretoras
- Relatórios e projeções personalizadas

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (backend)
- Capacitor (mobile)
